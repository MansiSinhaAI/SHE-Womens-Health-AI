# SHE-Womens-Health-AI

- AI-powered women’s health platform designed to simplify blood report understanding, symptom conversations, and personalized wellness insights.

## Problem Statement

- Many women struggle to understand medical reports, vitamin deficiencies, hormonal indicators, and wellness-related symptoms due to complex medical terminology and lack of accessible guidance.
- SHE aims to provide a safe, conversational, and AI-powered experience to help users better understand their health reports and wellness patterns.

## Features

- AI-powered health conversation interface
- Blood report upload and analysis
- OCR-based medical value extraction
- Simplified health explanations
- Personalized wellness insights
- Secure and privacy-focused architecture
- Modern responsive UI
